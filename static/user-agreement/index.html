8.1保证金

8.1.1 保证金包含分时保证金和长租保证金，用户使用车辆前应通过现金、刷卡、微信、支付宝或网银转账等方式向“一步用车”支付保证金。

8.1.2分时保证金使用规则：

8.1.2.1在芝麻信用达到“一步用车”活动要求后，会员可享受减免保证金服务。

8.1.2.2分时保证金优先用于当车辆发生损坏、违章以及发生行政处罚而产生费用后，用户不按时承担相应责任，“一步用车”有权从保证金中直接抵扣相应费用。保证金依据本协议被扣除后，会员在补足保证金后方可继续租用车辆。

8.1.2.3用户还车后，会员可在还车15个工作日后可通过软件平台向“一步用车”自助申请退还保证金。“一步用车”接受客户退还申请后，将在1-3个工作日内审查所有违章是否均已处理，所有违约及赔偿责任是否均已承担，上述责任均已承担后，“一步用车”将通过软件平台向客户退还保证金。

8.1.3长租保证金使用规则：

8.1.3.1会员在用车前需交纳租车押金，租车押金优先用于短/长租租车费用，如果产生车辆损失费后，用户不按时承担相应责任，“一步用车”有权从租车押金直接抵扣相应费用。

8.1.3.2在会员归还车辆后，“一步用车“将在1-3个工作日内审查车辆是否有损坏，在确定车辆无损坏或已承担相应责任后，“一步用车“将扣除本次租车费（如产生车辆损失的需扣除车辆损失）后通过软件向客户退还租车押金。

8.1.3.3用户还车时需缴纳违章押金,违章押金优先用于当车辆发生违章以及行政处罚而产生费用后，用户不按时承担相应责任，“一步用车”有权从保证金中直接抵扣相应费用。

8.1.3.4 用户还车后，会员可在还车后15个工作日后通过软件平台向“一步用车”自助申请退还。“一步用车”接受客户退还申请后，将在1-3个工作日内审查所有违章是否均已处理，所有违约及赔偿责任是否均已承担，上述责任均已承担后，“一步用车”将通过软件平台向客户退还违章押金。

8.1.4保证金为无息退还。

第九条 预约、取、还车说明

9.1 预定说明：会员可以通过APP客户端预约取车。

9.2 取车说明：

9.2.1 取车时，会员仔细观看车辆情况及周边情况，请确保车辆可以正常驾驶后再驶离站点，如果发生车身连接充电枪，需拔掉充电枪后，方可驶离。如果发生由于会员操作不当导致车辆或电桩损坏，产生费用由会员进行承担。

9.2.2应仔细检查确认车辆是否完好（包括但不限于检查车辆外观是否完好无损，车辆车窗户、车灯、车镜等是否有损毁，车辆轮胎是否有亏气、鼓包、起皮破损等异常问题），若发现车辆存在上述异常问题或存在有影响驾驶的，请不要驾驶该车辆离开，并留存照片或视频证据，同时立即拨打客服电话说明情况，否则即视为您提取的车辆完好无损。

9.2.3 取车前，须先确认车辆电量是否满足本次出行里程要求，若电量不足，请不要选择或驾驶离开， 因会员未确定电量或因估算不足，造成行驶过程中电量耗尽引发的拖车救援费用，以及由此造成的一切经济损失及法律责任均由会员承担。车辆的续航里程，表示在理想状况下可达公里数，实际续航里程会因温度、交通状况、驾驶习惯等因素不同。

9.2.4会员在使用车辆时驾照还在实习期的，需要按照交通部门相关法律法规要求，主动张贴实习标。如果由于会员违反相关法律法规所造成的经济损失及法律责任均由会员承担。

9.3 还车说明：

9.3.1 分时会员还车时应将车辆开回至指定还车位置，还车时间以会员实际操作完成还车流程的时间为准。目前分时服务只接受3天以内的订单，超过3天的订单需重新下订单租用，如超过3天未归还车辆“一步用车”有权收回车辆，由此产生的违约责任由承租人承担。

9.3.2短/长租会员还车时，需按订单提示在交纳违章押金后才能结束用车。

9.3.3会员在使用短/长租业务时，如果租用了充电枪、延长线、转化接头等辅助物料，请在确认物料归还后才能结束用车。

9.3.4短/长租提前还车：会员提前退租，应承担违约责任，违约金为未履行合同期限租金的20%。由于租赁车辆投保租赁期限内的保险，如乙方提前退租，未履行合同期限内的保险费用，甲方不予退还。

9.3.5 车辆（连同其钥匙和所有的附属物品，包括行驶本，手册，三脚架、灭火器、车贴、其他车载物品等）必须一并归还，并且要保持和租用期开始前一致的状况。如发生丢失或损坏需照价赔偿，相应产生的停运损失费及其它费用（包含但不限于）需一并赔偿。

9.3.6 会员应在归还车辆时，确保车内无任何个人财物。如还车时，有任何会员的个人财物落在车中，所产生的相应责任由会员个人承担，“一步用车”不承担任何责任。

第十条 事故和报告/责任和赔偿

“一步用车”向会员提供的车辆都已投保了交通责任强制险、车损险、第三者责任险、司机乘客责任险，不包含盗抢险，划痕险，玻璃单独破碎险，涉水险等附加险。

10.1 会员同意在任何时间均谨慎地使用车辆。会员在使用车辆期间发生任何事故，无论所处环境或严重程度如何，应立即联系“一步用车”客服人员。对于在汽车使用期间发生的任何保险理赔的调查和解决，会员有责任全力配合“一步用车”或其保险公司的工作。 如会员未能立即报告事故和配合事故调查、解决工作，则购买的不计免赔服务失效，会员应承担全部损失责任和费用。

10.2 会员下单租车时，因交通事故在理赔额度范围内，且保险公司没有拒赔或免赔的情况下，会员不需要承担车辆维修费用。如果因维修车辆，车辆停运超过3天，需承担车辆停运损失费，损失费按照对应车辆的日租封顶价计算。

10.3对保险责任以外的原因造成的事故损失以及保险不能理赔的事故损失项目，由会员全额承担。

10.4如果车辆在租用期内（或在服务期以外的，会员仍然占有或负责车辆的任何时段）被盗，会员必须在得知情况后尽快电话通知“一步用车”和警方并配合“一步用车”进行后续处理。
